NETWORK COMPUTING

Our Courses

Day 1: INTRODUCTION TO NETWORK SECURITY

An overview of Network Security fundamentals, emphasizing the need to protect Confidentiality, Integrity, and Availability (CIA) of information.

Day 2: NETWORK FUNDAMENTALS

Covers the core networking concepts including Internet Protocol (IP) addressing, ports, protocols, Open Systems Interconnection (OSI) and Transmission Control Protocol/Internet Protocol (TCP/IP) models, and data packet flow.

Day 3: LINUX BASICS

Introduces essential Linux commands, file system permissions, and networking utilities relevant for system administration and security tasks.

Day 4: FOOTPRINTING AND SCANNING

Explains how attackers collect target information through footprinting, and how scanning tools are used to identify system vulnerabilities.

Day 5: VULNERABILITY ASSESSMENT

This module teaches methods to detect, analyze, and prioritize system weaknesses to prevent exploitation.

Day 6: FIREWALL, INTRUSION DETECTION SYSTEM (IDS), INTRUSION PREVENTION SYSTEM (IPS) AND NETWORK DEFENSE

Covers key defense mechanisms including Firewalls, Intrusion Detection Systems (IDS), and Intrusion Prevention Systems (IPS) to protect against cyberattacks.

MINI PROJECT (NETWORK SECURITY)

A practical exercise involving penetration testing or network defense configuration to apply learned security concepts.
